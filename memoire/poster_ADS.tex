% === BLOC 1 : Contexte et problématique ======================
\block{Bloc 1 --- Contexte et problématique}{

    La loi République numérique (2016) impose l'ouverture des données publiques.
    Pourtant, leur exploitation reste inaccessible aux citoyens sans compétences techniques.
    Les LLM généralistes (ChatGPT, Claude) présentent quatre limites~: données obsolètes,
    absence de contextualisation territoriale, vocabulaire administratif non maîtrisé
    et coût récurrent incompatible avec les collectivités.

    \vspace{10pt}
    \begin{center}
    \begin{tikzpicture}
        \node[draw=cesiAccent, line width=3pt, rounded corners=10pt,
              inner sep=14pt, fill=white, text width=32cm, align=center,
              font=\Large\itshape\color{cesiBlueDark}] {
            Comment créer un système d'IA spécialisé dans les données publiques\\
            territoriales qui dépasse les limitations des LLM généralistes\\
            pour favoriser l'engagement citoyen~?
        };
    \end{tikzpicture}
    \end{center}
}

% === BLOC 4 : Conclusion et perspectives ======================
\block{Bloc 4 --- Conclusion et perspectives}{

    \textbf{Réponse à la problématique~:}
    Avec du matériel grand public (RTX 4070 Ti, 12\,GB) et un budget de 0\,€,
    le fine-tuning QLoRA de Mistral 7B surpasse la baseline de 87{,}5\,\%
    sur les données DECP/RNE et atteint 83{,}3\,\% de précision factuelle.
    Le modèle spécialisé est un \textbf{complément local, gratuit et souverain}
    aux LLM généralistes pour les collectivités territoriales.

    \vspace{10pt}
    \begin{minipage}[t]{0.48\linewidth}
        \textbf{\color{resultOrange}Limites~:}
        \begin{itemize}[leftmargin=12pt, itemsep=3pt]
            \item Catastrophic forgetting non mesuré
            \item 9/101 départements couverts
            \item Pas de validation humaine
            \item Vitesse $-$47\,\% (LoRA non fusionné)
        \end{itemize}
    \end{minipage}%
    \hfill
    \begin{minipage}[t]{0.48\linewidth}
        \textbf{\color{resultGreen}Perspectives~:}
        \begin{itemize}[leftmargin=12pt, itemsep=3pt]
            \item Benchmark MMLU-FR post-FT
            \item Intégration RAG (FAISS)
            \item Extension géographique nationale
            \item Interface Streamlit pour citoyens
        \end{itemize}
    \end{minipage}
}
